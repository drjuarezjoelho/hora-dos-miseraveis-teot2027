# Brainstorm de Design — HORA DOS MISERÁVEIS — TEOT 2027

## Contexto
Site de treinamento de elite para residentes em ortopedia e traumatologia, focado na aprovação no TEOT 2027. O projeto transmite seriedade acadêmica, urgência, pressão e alta performance. Cores principais sugeridas: vermelho, preto, branco.

## Ideia 1 — "Bunker Cirúrgico" (Brutalismo Médico)
<probability>0.07</probability>

<text>
**Design Movement:** Brutalismo digital com influências de design militar/tático

**Core Principles:**
1. Tipografia crua e impactante — blocos de texto densos com hierarquia agressiva
2. Estrutura de "briefing de guerra" — informação organizada como painéis de comando
3. Contraste extremo — preto absoluto com vermelho sangue e branco cirúrgico
4. Ausência de decoração supérflua — cada pixel serve a um propósito funcional

**Color Philosophy:**
- Preto absoluto (#0A0A0A) como base — representa a pressão, o desconhecido, o desafio
- Vermelho sangue (#DC2626) como acento — urgência, erro, alerta, a linha entre aprovação e reprovação
- Branco cirúrgico (#F5F5F5) para texto principal — clareza sob pressão
- Cinza aço (#374151) para elementos secundários — instrumentação, frieza técnica

**Layout Paradigm:**
- Grid assimétrico com painéis sobrepostos, como monitores de centro cirúrgico
- Seções com bordas duras, sem border-radius
- Barras laterais de informação tipo "status bar" com dados do cronograma
- Scroll vertical longo com seções que "entram" como slides de briefing

**Signature Elements:**
1. Linhas vermelhas horizontais que cortam a página como marcadores de bisturi
2. Contadores regressivos para o TEOT 2027 em destaque
3. Blocos de citação estilizados como "alertas de protocolo cirúrgico"

**Interaction Philosophy:**
- Hover revela informação adicional como "abrir prontuário"
- Cliques são decisivos — sem animações suaves, transições rápidas e cortantes
- Feedback visual imediato e direto

**Animation:**
- Elementos entram com slide lateral rápido (150ms), como instrumentos sendo passados
- Texto aparece com efeito de digitação em seções-chave
- Barras de progresso preenchem com urgência
- Sem bounce, sem ease-out suave — tudo é snap e precision

**Typography System:**
- Display: "Oswald" (bold, uppercase) — impacto militar, títulos que gritam
- Body: "Source Code Pro" ou "JetBrains Mono" — precisão técnica, leitura de protocolo
- Accent: "Barlow Condensed" — dados, números, estatísticas
</text>
</response>

## Ideia 2 — "Sala Escura" (Noir Acadêmico)
<probability>0.06</probability>

<text>
**Design Movement:** Dark academia meets editorial médico de alto padrão

**Core Principles:**
1. Elegância sombria — sofisticação que intimida sem ser agressiva
2. Hierarquia editorial — como uma revista médica de elite em formato digital
3. Espaço negativo como ferramenta de pressão — o vazio comunica exigência
4. Tipografia como protagonista — letras que comandam a página

**Color Philosophy:**
- Negro profundo (#0C0C0C) — a sala de estudos às 3h da manhã, onde se forja a excelência
- Vermelho bordô (#991B1B) — sangue seco, gravidade, consequência
- Marfim (#FFFBEB) para texto — páginas de tratados clássicos de ortopedia
- Dourado escuro (#92400E) para destaques — o título que se conquista

**Layout Paradigm:**
- Layout editorial com colunas assimétricas inspirado em revistas como JBJS
- Hero sections com tipografia oversized ocupando 70% do viewport
- Cards com bordas finas e sombras profundas, como páginas de livro empilhadas
- Seções divididas por linhas finas douradas

**Signature Elements:**
1. Números de seção oversized (estilo editorial) em vermelho bordô como marcadores
2. Aspas tipográficas gigantes em citações motivacionais/técnicas
3. Ícones de ossos/articulações estilizados em line-art minimalista

**Interaction Philosophy:**
- Hover com transições elegantes (300ms ease) — como virar página de atlas
- Parallax sutil em imagens de fundo
- Revelação progressiva de conteúdo ao scroll

**Animation:**
- Fade-in com translate-y sutil (20px) nas seções
- Underline animado em links que se expande da esquerda
- Números do contador incrementam com easing suave
- Transições de opacidade em cards ao entrar no viewport

**Typography System:**
- Display: "Playfair Display" (bold) — autoridade clássica, peso acadêmico
- Body: "Lora" — legibilidade em textos longos, herança editorial
- Accent: "Bebas Neue" — números, contadores, dados de impacto
</text>
</response>

## Ideia 3 — "Centro de Comando" (Tech Médico Tático)
<probability>0.08</probability>

<text>
**Design Movement:** Interface de comando tático com estética de HUD médico futurista

**Core Principles:**
1. Dashboard de guerra — cada seção é um painel de monitoramento
2. Dados em primeiro plano — métricas, contadores, progresso sempre visíveis
3. Vermelho como sistema de alerta — usado com precisão cirúrgica para chamar atenção
4. Geometria angular — cortes diagonais e linhas que sugerem precisão e velocidade

**Color Philosophy:**
- Preto grafite (#111111) — a tela de um monitor de centro cirúrgico desligado
- Vermelho alerta (#EF4444) — o sinal que não pode ser ignorado
- Branco frio (#E5E7EB) — dados no monitor, informação pura
- Verde cirúrgico (#065F46) como acento terciário — o campo estéril, a zona de ação

**Layout Paradigm:**
- Seções com cortes diagonais (clip-path) criando sensação de velocidade
- Cards com cantos cortados (estilo HUD) em vez de border-radius
- Sidebar fixa com navegação tipo "painel de instrumentos"
- Grid modular com gaps precisos

**Signature Elements:**
1. Linhas de scan horizontais semi-transparentes que cruzam seções (como monitor cardíaco)
2. Badges de nível de dificuldade estilizados como classificação militar
3. Barra de progresso do cronograma sempre visível no topo

**Interaction Philosophy:**
- Hover com glow vermelho sutil — como indicador de ativação
- Transições com timing preciso (200ms) — nem lento, nem instantâneo
- Scroll-triggered animations que "ativam" painéis sequencialmente

**Animation:**
- Elementos entram com scale de 0.95 para 1 + fade (200ms)
- Linhas de scan animadas continuamente em background
- Contadores numéricos com efeito de "rolling numbers"
- Barras de progresso com animação de preenchimento gradual

**Typography System:**
- Display: "Rajdhani" (bold) — futurista, técnica, angular
- Body: "IBM Plex Sans" — clareza técnica, herança de sistemas
- Accent: "Orbitron" — números, contadores, dados de HUD
</text>
</response>

## Decisão

**Abordagem escolhida: Ideia 1 — "Bunker Cirúrgico" (Brutalismo Médico)**

Justificativa: Esta abordagem é a que melhor traduz a filosofia do projeto HORA DOS MISERÁVEIS — sem concessões, sem conforto, sem decoração. O brutalismo digital com elementos militares/táticos comunica exatamente o que o projeto propõe: pressão extrema, precisão cirúrgica e formação de elite. A ausência de border-radius, as transições cortantes e a tipografia impactante criam uma experiência que já começa a "treinar" o residente desde o primeiro acesso ao site.
